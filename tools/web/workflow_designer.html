Node
Kéo node vào canvas. Kéo từ cổng phải, thả lên node bất kỳ là tự nối. Nhóm: bấm nút ▢ trên thanh công cụ rồi kéo để vẽ khung (hoặc chọn vài node rồi bấm) — kéo tiêu đề khung để di chuyển cả nhóm, bấm đúp để đổi tên. Với node Lặp lại: thả vào cổng lặp (dưới) để nối vòng lại. Ctrl+C/V sao chép/dán · Ctrl+D nhân đôi · Delete xoá.
